Thanks, homies.

This website is almost entirely JS, CSS + HTML, with a couple of server-side pages (PHP) thrown in to handle conversion of eBird API data into JSON. The following is a list of all the open source scripts and resources used. A big thanks to all the developers for your work. You rock. May your first child be a masculine child, and all that.

  • RequireJS - JS AMD dependency loader
  • jQuery
  • jQuery UI
  • jQuery tablesorter
  • html5shiv
  • Modernizr
  • Underscore
  • Google Maps inverted circle
  • Bootstrap, modals and transition
  • Bootstrap, cyborg theme (heavily modded)
  • Sass
  • Spinners - my own project fork

Lastly, a huge thanks to the Cornell Lab of Ornithology for creating their public API, which makes this site possible.
