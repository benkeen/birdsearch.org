Thanks!

This website is almost entirely JS, CSS + HTML with a couple of server-side pages (PHP) to handle conversion of eBird API data into JSON. This script was built using the following open source scripts and resources. A big thanks to all the developers for their work.

  • RequireJS - JS AMD dependency loader
  • jQuery
  • jQuery UI
  • jQuery tablesorter
  • html5shiv
  • Modernizr
  • Underscore
  • Google Maps inverted circle
  • Bootstrap, modals and transition
  • Bootstrap, cyborg theme (heavily modded)
  • Sass
  • Spinners - my own project fork

As always, thanks to Addy Osmani for his many articles and books on JS architecture. And lastly, a huge thanks to the Cornell Ornithology lab for their API, without which this site would be impossible.
